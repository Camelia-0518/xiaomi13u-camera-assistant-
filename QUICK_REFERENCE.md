
## ⭐ 星空参数速查

### 银河拍摄
```
焦距: 23mm
光圈: f/1.9
ISO: 1600-3200
快门: 15-25秒 (超过25秒星星拖线)
对焦: 手动无限远
格式: RAW
```

### 避免拖线公式
```
最大快门 = 500 ÷ 焦距

23mm镜头: 500÷23 ≈ 21秒
```

## 🎯 故障排除

### 照片模糊
- [ ] 快门速度 ≥ 1/焦距 (如75mm需≥1/75s)
- [ ] 检查对焦位置
- [ ] 开启防抖
- [ ] 使用三脚架

### 颜色偏色
- [ ] 调整白平衡
- [ ] 切换徕卡风格
- [ ] 检查环境光源

### 噪点过多
- [ ] 降低ISO
- [ ] 确保正确曝光
- [ ] 使用夜景模式代替高ISO
