# 小米13 Ultra 摄影助手 - Android自动检测版

真正的全自动方案：打开相机 → 自动弹出拍摄指导悬浮窗

## 🎯 功能特点

- ✅ **全自动检测**：通过Android无障碍服务监听相机启动
- ✅ **悬浮窗指导**：覆盖相机界面，不干扰拍摄
- ✅ **6大场景**：人像、夜景、街拍、风景、美食、星空
- ✅ **一键参数**：点击场景按钮显示详细参数
- ✅ **最小化**：可缩为小圆点，随时展开
- ✅ **街拍快捷键提示**：双击音量下键快速启动

## 📱 系统要求

- Android 8.0 (API 24) 或更高
- 小米13 Ultra（其他小米手机也可使用）
- 无需Root

## 🚀 安装方式

### 方式1: 直接安装APK（最简单）

1. 下载 `xiaomi13u-camera-assistant.apk`
2. 安装APK（允许未知来源）
3. 打开APP，按提示开启权限
4. 打开相机，享受自动指导！

### 方式2: 自行编译

```bash
cd android-auto-camera

# 使用Android Studio打开项目
# 或使用命令行编译

./gradlew assembleDebug

# APK输出位置
# app/build/outputs/apk/debug/app-debug.apk
```

## 📖 使用指南

### 首次设置

1. 打开「小米13U摄影助手」APP
2. 点击「开启无障碍服务」
3. 在系统设置中找到「小米13U摄影助手」
4. 开启服务，返回APP
5. 授予悬浮窗权限
6. ✅ 完成！

### 日常使用

1. **打开相机APP**
   - 屏幕顶部自动弹出拍摄指导
   
2. **选择场景**
   - 点击对应场景按钮（人像/夜景/街拍等）
   - 显示详细参数卡片
   
3. **查看参数**
   - 焦距、光圈、ISO、快门速度
   - 按提示设置相机
   
4. **最小化**
   - 点击「−」按钮缩为小圆点
   - 点击小圆点恢复
   
5. **关闭**
   - 点击「×」或退出相机自动关闭

### 街拍快捷键

**锁屏状态双击音量下键** → 0.8秒快速抓拍

悬浮窗会显示此提示，方便记忆。

## 🐛 故障排除

### 悬浮窗不显示

1. 检查悬浮窗权限是否开启
2. 检查无障碍服务是否运行
3. 尝试重启APP

### 检测不到相机

1. 确保使用小米自带相机
2. 检查无障碍服务是否被系统杀掉
3. 在电池优化中设为「不优化」

### 悬浮窗遮挡取景

1. 点击「−」最小化为小圆点
2. 或拖动悬浮窗到其他位置（如需要实现拖动功能）

## 📝 技术细节

### 检测原理

使用Android无障碍服务监听 `TYPE_WINDOW_STATE_CHANGED` 事件：

```java
@Override
public void onAccessibilityEvent(AccessibilityEvent event) {
    if (event.getPackageName().equals("com.android.camera")) {
        showFloatingGuide();  // 显示指导
    }
}
```

### 支持的相机APP

- 小米相机 (`com.android.camera`)
- MIUI相机 (`com.miui.camera`)
- Google相机 (`com.google.android.GoogleCamera`)
