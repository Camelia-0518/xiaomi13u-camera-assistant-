---

### 方案A：Android无障碍App（⭐⭐⭐ 全自动推荐）

**效果**：打开相机 → 自动弹出拍摄指导悬浮窗

```bash
# 方式1: 直接安装APK（推荐）
# 下载地址: releases/xiaomi13u-camera-assistant.apk

# 方式2: 自行编译
cd android-auto-camera
./gradlew assembleDebug
# 安装 app/build/outputs/apk/debug/app-debug.apk
```

**设置步骤**：
1. 安装APK
2. 打开APP → 开启「无障碍服务」
3. 授予「悬浮窗」权限
4. 打开相机，自动显示指导！

**回退**：设置 → 应用 → 卸载（零残留）

---

### 方案B：Tasker+Termux（⭐⭐☆ 已有Tasker用户）

```bash
# 运行自动配置脚本
./setup_auto_detect.sh
# 选择选项 1 (Tasker方案)
```

**手动设置**：
1. 安装 [Tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm)
2. Tasker → 任务 → + → 「显示拍摄提示」
3. 添加操作 → 插件 → Termux → Termux:Task
4. 可执行文件: `tasker_integration.sh`
5. 配置文件 → + → 应用 → 相机
6. 选择刚才创建的任务

---

### 方案C：Termux自启动（⭐⭐☆ 免费方案）

```bash
# 运行配置脚本
./setup_auto_detect.sh
# 选择选项 2 (Termux方案)

# 或者手动配置
mkdir -p ~/.termux/boot
cat > ~/.termux/boot/start-camera-monitor << 'EOF'
#!/data/data/com.termux/files/usr/bin/sh
termux-wake-lock
cd ~/xiaomi13u-camera-assistant
python3 auto_detector.py --monitor &
EOF
chmod +x ~/.termux/boot/start-camera-monitor
```

## 📱 使用示例

### CLI 交互式使用

```
🎯 请输入场景或命令: 人像

============================================================
👤 人像 - 拍摄人物，强调肤色和背景虚化
============================================================

📷 推荐模式: 专业模式或人像模式

⚙️ 参数设置:
   焦距      : 75mm 长焦镜头 (3.2x)
   光圈      : f/1.9 (最大虚化)
   ISO       : 100-400 (根据光线)
   快门      : 1/125s 或更快
   白平衡    : 自动 或 日光/阴天
   对焦      : 眼睛对焦
   徕卡风格  : 徕卡生动 (肤色更暖)
   影调      : +5 至 +10 (增加对比)
   锐度      : +5 (保持自然)

🎨 构图技巧:
   1. 使用 75mm 长焦压缩背景
   2. 光圈 f/1.9 获得奶油般虚化
   3. 人物占画面 1/3 到 1/2
   4. 眼睛放在上三分线
   5. 利用街道、门窗作为框架

📏 推荐距离: 1.5-3米 (75mm镜头最佳距离)
============================================================
```

## 🎓 进阶技巧

### 徕卡色彩玄学
- **原理**：徕卡对红、黄、蓝三原色敏感
- **应用**：拍摄时寻找红墙、蓝天、黄叶

### 可变光圈选择
- **f/1.9**：最大虚化，人像、特写
- **f/4.0**：最佳画质，风景、建筑

### 街拍模式快捷键
**锁屏状态双击音量下键** → 0.8秒启动并拍照
